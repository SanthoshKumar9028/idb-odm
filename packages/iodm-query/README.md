# iodm query package

## Overview

The `iodm-query` package provides a powerful query interface for IndexedDB operations using the IDB-ODM (IndexedDB Object Data Mapper). It enables developers to build complex queries with an intuitive API for filtering, sorting, and retrieving data from IndexedDB stores.

### Features

- Fluent query builder API
- Advanced filtering and sorting capabilities
- Type-safe query operations
- Support for compound queries
- Optimized performance for large datasets
